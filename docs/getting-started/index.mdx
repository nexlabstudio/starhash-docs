# Getting Started

Welcome to StarHash! This guide will help you get up and running quickly.

## What You'll Learn

1. **Create an Account** - Sign up for StarHash
2. **Quickstart** - Build and test your first USSD flow
3. **Core Concepts** - Understand flows, nodes, and sessions

## How It Works

1. **Design** - Create USSD flows visually in Studio
2. **Test** - Simulate sessions before deployment
3. **Deploy** - Publish flows to your shortcode
4. **Monitor** - Debug sessions with X-Ray console

## Prerequisites

Before you begin, make sure you have:

- A modern web browser (Chrome, Firefox, Safari, Edge)
- An email address for account registration
- A USSD shortcode from a supported provider (for production deployment)

<Info>
  New to USSD? Check out [Core Concepts](/getting-started/concepts) to learn the fundamentals.
</Info>

## Create Your Account

1. Go to [StarHash Studio](https://studio.starhash.dev)
2. Click **Sign Up**
3. Register with Google, GitHub, or email
4. Verify your email address
5. You're ready to build!
