# Changelog

## December 2025

### v2.4.0 - December 20, 2025

**New Features**
- Added Nalo Solutions provider support for Nigeria
- Introduced flow templates for common use cases
- Added bulk session export (Pro and Enterprise)

**Improvements**
- Improved simulator response time by 40%
- Enhanced X-Ray debugger with variable timeline
- Better error messages for validation failures

**Bug Fixes**
- Fixed issue where connections would disappear after undo
- Resolved OAuth redirect issues on mobile browsers
- Fixed timezone display in session history
