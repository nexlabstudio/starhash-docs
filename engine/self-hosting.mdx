# Enterprise Deployment

For organizations with specific compliance, security, or scale requirements.

## Deployment Options

### StarHash Cloud (Default)

The standard hosted solution:

- Fully managed infrastructure
- Automatic updates
- 99.9% SLA uptime
- Global CDN
- Included in all plans

### Dedicated Cloud

Your own isolated StarHash instance:

| Feature | Included |
|---------|----------|
| Isolated infrastructure | Yes |
| Custom domain | Yes |
| Dedicated support | Yes |
| Custom SLA | Yes |
| Data residency options | Yes |

### On-Premise

Deploy StarHash within your own infrastructure:

| Feature | Included |
|---------|----------|
| Full source access | License-based |
| Your infrastructure | Yes |
| Air-gapped deployment | Yes |
| Custom integrations | Yes |
| Professional services | Available |
